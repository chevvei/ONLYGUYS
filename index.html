年度清单

正文以仓库 data/year-todos.json 为准 · 勾选进度仅保存在本机

如何使用（编辑文件 → 推送 → 网页对照）
  1. 在本地用编辑器打开 data/year-todos.json，按年份填写 lists；每条保留稳定 id 字符串，便于本机记住是否勾选。
  2. 保存后执行 git add data/year-todos.json → git commit → git push。
  3. 在首页点 清单 打开此弹层，点 从仓库同步（或下次打开页面前已自动拉取）即可看到最新正文。
  4. 网页上的增删改、勾选会写入浏览器缓存；与仓库冲突时，以你 push 的文件 为准，同步后会合并同 id 的勾选状态。
  5. 详细说明与云端方案见仓库内 docs/todo-year-list.md。

暂无条目。可在上方新增年度，或编辑仓库 JSON 后点「从仓库同步」。

    本地备份

    换设备或清缓存前可导出；若要更新仓库，请把 JSON 覆盖 data/year-todos.json 再 push。
