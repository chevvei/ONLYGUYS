# Changelog

## 2026-03-29（第五十次更新）

- **展板随机模式**：`data/scroll-board.json` 新增 `randomMode`，设为 `true` 时底部文案随机无重复轮播（一轮播完再重置）。
- **音乐曲目替换**：浮动音乐改为网易云单曲随机（`type=2`）：邓紫棋《唯一》、李荣浩《恋人》、老狼/王婧《想把我唱给你听》、郭顶《我们俩》、苏打绿《小情歌》。

## 2026-03-29（第四十九次更新）

- **底部展板配置化**：新增 **`data/scroll-board.json`**（`speedSec` + `lines`），你只需改这个文件即可控制底部滚动文案与速度；`music-player.js` 会自动拉取并循环展示。
- **滚动效果**：底部条改为更慢速（默认 `42s`）+ 轻微亮度跳动（`lyric-bounce`），更像信息展板。
- **边缘修复**：音乐唱片在窗口边缘展开时，播放器面板会自动判断方向（左/右、上方），减少超出屏幕导致显示不全的问题。

## 2026-03-29（第四十八次更新）

- **音乐组件**：新增右下角 **浮动可拖拽唱片** + 展开播放器面板（`#musicFab`），接入 `js/music-player.js`；每次进入随机选择一条歌单链接（当前配置为偏抒情/缓慢/轻快英文风格的网易云歌单示例，可继续替换为你的网易云/QQ链接）。
- **滚动歌词带**：页面底部新增 **长条歌词滚动栏**（`#musicLyricBar`），以当前随机歌单主题句循环滚动展示，形成动感氛围。
- **样式**：`css/styles.css` 增加 `music-fab*` 与 `music-lyric-bar*` 规则；音乐唱片位置上移，避免覆盖底部歌词带。

## 2026-03-29（第四十七次更新）
