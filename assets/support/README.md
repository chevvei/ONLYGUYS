# 赞助 / 众筹素材（`assets/support/`）

静态页「**看世界**」弹层从这里读取收款码等图片，便于与业务代码分离、以后加渠道也不乱。

## 当前文件

| 文件 | 用途 |
|------|------|
| **`wechat-pay.jpg`** | 微信收款码（弹层主图） |

更换收款码时：**直接覆盖同名文件**即可，无需改 HTML（除非你改了文件名）。

## 扩展建议（以后）

- 可增加 **`alipay-pay.jpg`** 等，再在 `index.html` 的 `#modalSupport`（或同类弹层）里加第二张图与说明文案。
- 若渠道变多，可改为 **`content/support.md`** 或小型 JSON 由脚本渲染；当前项目为保持零构建，先用「一图一文件 + README 约定」即可。

## 隐私与安全

收款码会随站点 **公开可访问**（与任意静态图片相同）。勿将含敏感信息的截图误传至此目录。
