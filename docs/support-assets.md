# 众筹 / 赞助素材说明

## 目录

所有赞助相关**静态图**放在 **`assets/support/`**，与 `creators/`、`onlyGuysPic/` 等并列，避免堆在项目根目录。

详见该目录下的 **[`assets/support/README.md`](../assets/support/README.md)**。

## 页面行为

- 顶栏 **「订阅」右侧** 为 **「看世界」**，与 **发现 / 创作者 / 关于** 一样通过 **`data-open-modal`** 打开弹层。
- 弹层内含 **标语**、**「看世界打卡」** 地图（**中国 / 世界** 底部深色双栏切换；**中国** 为雄鸡示意国界 + 加粗岸线 + 陆块渐变；**世界** 为 **正射投影球形地球**、圆形裁切 + 海洋径向渐变 + 经纬网 + 陆块示意）、微信收款码 **`assets/support/wechat-pay.jpg`**。**仅「中国」模式** 显示「已点亮 · 城市列表」图例；**「世界」模式** 不展示该列表。更新收款码：覆盖该文件并推送即可。

## 旅行地图与扩展

- 逻辑与坐标在 **`js/travel-map.js`**：`CITIES` 可带 **`labelDx` / `labelDy`** 微调图上文字相对光点的偏移。
- 中国视图为全国等距圆柱投影框；含 **大陆主轮廓 + 海南 + 台湾** 三路径（简化示意）；世界视图为东亚聚焦 + 圆形裁切。
- **交互**：**中国** — 滚轮缩放、拖拽平移、双击重置；**世界（球）** — 拖拽旋转、滚轮缩放视距、双击重置。**不接** 第三方在线地图瓦片（无 API、零构建）；若需真实道路/卫星图，需自行接入合规地图 SDK 并处理 Key 与审图号。
- 可选：在 **`assets/travel/`** 放置 GeoJSON 并由脚本 `fetch` 解析（见该目录 **README**）。

## 扩展

在 **`index.html`** 中搜索 **`modalSupport`**，可在此弹层内增加第二收款方式、短文案或外链；大图建议继续放在 **`assets/support/`** 并沿用「语义化文件名」。
