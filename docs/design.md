# OnlyGuys 设计说明（雏形）

## 目标

- **现代、有记忆点**的社交/创作者向落地页外观。
- 当前视觉**偏 OnlyFans 一类**：深色界面、**青蓝强调色（约 #00AFF0）**、简洁顶栏与圆角主按钮，而非高对比橙色条主导。
- **仅借鉴**常见订阅平台的布局与配色习惯，用于合法的品牌/产品展示或学习用途。
- **全端可读**：手机、iPad、桌面使用同一套流式排版与 `clamp()` 字号。

## 技术约束（GitHub Pages）

- 无自建服务端：主背景图为 **`onlyGuysPic/cv.jpg`** / **`onlyGirlsPic/Qing.jpg`**，由主题切换；访客不能改服务器上的资源。
- 「关于」通过 `fetch` 读取 `**content/about.md**`；图库通过 `fetch` 读取 `**js/gallery-manifest.json**`；根目录 `**.nojekyll**` 用于关闭 Jekyll，保证上述静态路径可访问。
- 评论通过 **Giscus** 走 GitHub Discussions，配置见 `docs/operations-manual.md`。
